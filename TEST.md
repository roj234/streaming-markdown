

### 公式测试向量之一

| 场景 | Markdown 源码 | 渲染结果 | 说明 |
| :--- | :--- | :--- | :--- |
| **未转义** | `The price is $100.` | *The price is $100.* (可能被高亮或变斜) | 被识别为公式开始，直到下一个 `$`。如果后面没闭合，会报错或格式错乱。 |
| **转义符号** | `The price is \$100.` | The price is $100. | **最推荐**。显示正常，无特殊样式。 |
| **代码块** | `The price is `$100`.` | The price is `$100` | 变成等宽字体，通常有背景色。 |
| **混合写法** | `Total: \$50 + \$20 = \$70` | Total: $50 + $20 = $70 | 多个价格混排时的标准写法。 |
